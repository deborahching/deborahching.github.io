Website Redesign

The aim of this project is to redesign the website of LCSD Museums, Traversing the Forbidden City. As activities among the website is targeting school kids and teachers, a new visual style which is more bright and joyful was created to attract target audience. 

View the NEW website View the original website


Hero image created for the website. Recreating flat illustration to replace the actual picture used by the original website, this modern style gives a feeling of bringing tradition in new age, to echo with the education purpose of website.

Colour scheme used for the website, extracted from the colours in the Forbidden City.

Menu Design

On the Menu, icons of different features are added on the right hand side, to create more convenient using experience for users while using the website.

Feature Design

On the Feature page, buttons for accessing to different pages contain flipping feature, words were hidden under, providing user a more interactive experience, and give an exploring feeling. 

Article page design

Interactive features had been added in the article pages. When users are scrolling the page, visual elements, the clouds, will be moved according to the scrolling path.
